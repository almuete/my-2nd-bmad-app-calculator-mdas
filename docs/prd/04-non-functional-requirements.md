# 4. Non-Functional Requirements
- Performance ≤1s load
- WCAG 2.2 AA compliance
- Consistent Tailwind theme tokens
- Componentized maintainable design

- Color contrast AA (≥4.5:1) for digit/decimal cyan keys across default/hover/active/focus
- Visible focus indicator on cyan keys (ring and ring-offset clearly perceptible)
- Dark theme parity for contrast and focus visibility (if dark mode enabled)
- No regression to operator/action key visuals
